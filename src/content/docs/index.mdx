Seyfert is a powerful framework to interact with the Discord API while providing a set of utilities that make it easier to build complex bots.

Some of it's features include:

- Fully written in TypeScript
- Decorators support [What the heck are decorators?](https://www.typescriptlang.org/docs/handbook/decorators.html)
- Lower memory usage
- Great developer experience
- Customizable
- And more!
